Instruções de como configurar o Projeto para integrar com o Eureka

Está tudo funcionando!

Configurando o serviço Eureka

O arquivo de configuração está localizado em conf/eureka-client.properties. Nesse arquivo devem ter as seguitnes cofnigurações

  1. Endereço do serviço rodando do eureka.
  2. Nome do projeto atual que será registrado no eureka.
  3. Porta que está sendo servido o projeto que está rodando.

Eureka Client

É configurado um módulo do EurekaClient para solicitar consulta a outros serviços que serão utilizados.

Você pode disponibilizar o client injetado seu recurso a uma classe. @Inject public StatusController(EurekaClient eurekaClient).

É importante implementar a rota de /Status utilizada como link no painel do serviço do Eureka.

Parando um serviço

O eureka entende que mesmo que ele perca a comunicação com o serviço ele ainda pode está rodando.

É importante configurar um hook para informar ao serviço do eureka que aquele serviço está sendo desligado.

Verifique a classe EurekaStartup para verificar um exemplo de como pode ser feito.
